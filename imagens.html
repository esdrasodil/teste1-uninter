Imagens

Para inserir imagens, utilizar a tag img, que é a abreviaçao de image. Dentro dessa tag, utilizar o src, que é abreviação de source, ou endereço onde a imagem está hospedada.

Para questões de acessibilidade, utilizar sempre a tag ALT para descrever a imagem em texto.

Imagens de praia

Para inserir imagens, podemos fazer colocando a tag IMG, seguido de SRC, que indica a fonte do link onde a imagem está hospedada e ALT, que é o texto que será exibido caso a imagem não possa ser exibida.

Formato: < img src = "link da imagem.jpg" alt = "descrição da imagem" >
