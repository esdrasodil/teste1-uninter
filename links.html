Links

Para inserir um link, é necessário utilizar a tag a, que significa anchor.

Existem diversos tipos de links, como veremos a seguir.

Links externos

Servem para qualquer site na internet. Deve ser inserido o protocolo usado no site (HTTP ou HTTPS) no começo.

Por exemplo, utilize https://www.google.com ao invés de apenas google.com

  • Site do Google
  • Site do Facebook
  • Site do YouTube

Links Internos

Para chamar páginas do mesmo site, localizados na mesma pasta que este documento.

  • Voltar para home.
  • Ir até as listas.

Links quebrados

Levam para lugar nenhum porque o endereço não existe, independentemente do motivo.

Nesse caso, mudam de cor após o primeiro acesso.

  • Para acessar um link quebrado, clique aqui.
