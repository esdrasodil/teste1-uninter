Tabelas

Para criar o título da tabela, utilizar a TAG TH, que significa Table Heading.

Para criar uma linha na tabela, utilizar a TAG TR, que significa Table Row.

Para criar uma nova célula, que também pode ser interpretada por coluna, utilizar a TAG TD, que significa Table Data.

Vamos treinar a criação de uma tabela...

Depois, conter a tag TR para inserir uma linha.
Dentro da linha, inserir a tag TH, quando quiser adicionar uma coluna de título e...
TD, quando quiser adicionar uma celula comum.
